# M5StickC PLUS2 Rust

Rust project for the M5StickC PLUS2, a compact ESP32 development board with built-in display, IMU, and battery.

## Quick Start

```bash
source ~/export-esp.sh

cargo run --package stick --release

espflash monitor

# simulator
# https://crates.io/crates/embedded-graphics-simulator
brew install sdl2
export LIBRARY_PATH="$LIBRARY_PATH:$(brew --prefix)/lib"

cargo +stable run --package sim --target aarch64-apple-darwin
```
